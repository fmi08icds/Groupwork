## Instructions for running the program

1. Make sure you downloaded the required dataset from [kaggle.com](https://www.kaggle.com/datasets/paultimothymooney/chest-xray-pneumonia)
2. put the folder including the images into the data/ folder
3. Make sure the required libraries are installed in your environment
4. Run the script with the command: **python3 src/main.py**
5. You can change the input arguments:

- **-m naive** for using the naive cnn
- **-m resnet** for using the resnet
- **-m inspect** for having a look on a sample image
